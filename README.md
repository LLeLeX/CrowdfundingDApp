# Crowdfunding DApp
Crowdfunding DApp Based on Node JS and Ethereum

# 概述

区块链众筹DApp基于Node JS和以太坊平台。使用React框架开发客户端，Express框架开发服务器，Solidity开发智能合约。

# 功能

* 项目发布模块
* 项目跟投模块
* 资金去向模块
* 付款请求模块

# 运行

IntelliJ IDEA 2019.2 x64

```
//进入文件夹
cd crowdfunding，cd crowdfundingServer

//安装相关package
npm init

//客户端运行
npm start

//服务器运行
node server
```
